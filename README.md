# backend-gongcha

교통사고 과실비율 책정 서비스
-------------
교통사고 후 보험처리를 하는데, 본인 보험사의 과실 산정이 마음에 안드는 경우가 종종 있습니다. 
하지만 개인은 분쟁을 위한 신뢰성 있는 근거를 마련하기는 쉽지 않습니다.
이를 위해 여러 전문가의 실시간 모니터링을 통한 과실비율책정 서비스를 제작했습니다.

프로젝트 기능
-------------
- 교통사고 영상 업로드
- 여러 보험사의 과실책정 서비스
- 교통사고에 대한 분석시트 제공

사용기술
--------
  - Java
  - SpringBoot
  - MySQL
